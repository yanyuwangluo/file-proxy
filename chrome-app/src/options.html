文件加速下载服务

支持命令行，可在 Git Bash 中 clone，或终端 wget 代下(详情见项目说明)

Github 项目请在 Code -> Download Zip 按钮右键复制链接，然后粘贴到下载框

CloudFlare 代下无速度及文件大小限制，本站服务器代下限制文件大小 2G

欢迎小伙伴们收藏和分享本服务，您的支持是我最大的动力~~~

Git 项目加速：git clone https://www.yanyu.ltd/< https 开头的项目地址>

Wget 加速下载：wget https://www.yanyu.ltd/<原有链接>

google 或 jcenter 仓库镜像：https://pd.zwc365.com/cfworker/<原有仓库地址>

加速任意链接：https://www.yanyu.ltd/<任意 http 或 https 链接>


在后面拼接的URL必须是 http:// 或者 https:// 开头

合格的URL: https://github.com/yanyuwangluo/file-proxy.git

不合格的示例: github.com/yanyuwangluo/file-proxy/archive/master.zip

不合格的示例: www.baidu.com
